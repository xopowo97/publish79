## 🔒 [보너스] 특허 기술 — 공모전에서 경쟁자와 차별화되는 1가지

> 다른 팀들도 공공데이터 + AI를 씁니다. 그런데 이것은 없습니다.

**"AI가 자율로 코드를 고치고 배포하는 과정에서 API Key가 유출되면 회사가 망합니다."**

출판친구는 이 치명적 리스크를 해결한 **세계 최초 실시간 AI 보안관 통합 아키텍처**를 특허 출원 중입니다:

- ① **1차 방어**: 공공데이터 수집 과정에서 프롬프트 인젝션 악성 패킷 실시간 차단
- ② **2차 방어**: 자가치유 코드 내 민감 자격증명 30초 전수 스캔 → 환경변수 자동 치환
- ③ **3차 방어**: 관리자 전용 세션 기반 접근 제어 (일반 계정의 통제실 URL 직접 접근 차단)
