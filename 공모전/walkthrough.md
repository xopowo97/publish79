## 2. 이번 회차 신규 구현: 이지퍼비터 원가 최적화 제안 카드

### ① [NEW] `checkPrintCostOptimization(bookData, printQty)` 함수
- **위치**: [control.js](file:///c:/Users/seo%20sang%20won/001.작업파일/004.%20출판친구/018.%20안티그래비티/control.js) — 섹션 12-2 (L873~)
- **트리거 조건**:
  - 제작 부수(`printQty`) **50부 미만** (디지털 연속지 전환 불가 임계치)
  - 도서 기본 사양: **신국판** (152×225mm, 낱장 2판거리 → 면당 12원)
- **원가 계산 로직**:
  - 신국판: 315×467mm 트레이 1장에 2판 거리 → 면당 **12원**
  - A5: 315×467mm 트레이 1장에 4판 거리 → 면당 **8원**
  - 절감율: **(12-8)/12 × 100 = 33.3%**
  - 총 절감액: `(면당12원 - 면당8원) × 총페이지 × 부수`
- **UI**: AI 헬퍼 패널(Antigravity)에 에메랄드그린 강조 제안 카드를 렌더링하며, 다음 정보를 포함합니다:
  - 제작 부수, 예상 페이지수
  - 신국판 인쇄비 vs A5 전환 인쇄비 2열 비교 그리드
  - 총 절감 예상액 및 권당 절감액
  - 폰트 임베디드 처리로 서버 폰트 불필요 안내
- **승인 플로우**: [✅ A5 판형 최적화 승인 및 시뮬레이션 가동] 버튼 클릭 → `ctrlApproveA5Optimization()` → `startCtrlSimByBookData()` 자동 트리거
- **감사 로그**: 3번(수익성 분석 팀장), 13번(오케스트레이터) 에이전트 감사 로그 DB 자동 기록

### ② [MODIFY] 파이프라인 완료 시 자동 호출 훅 연결
- **위치**: [control.js](file:///c:/Users/seo%20sang%20won/001.작업파일/004.%20출판친구/018.%20안티그래비티/control.js) L771~774

## 3. 시나리오 검증

### 시나리오: 이지퍼비터 원가 최적화 자동 발동
1. 에이전트 통제실에서 **🚀 파이프라인 실행** 클릭.
2. 파이프라인 완료 후 1초 뒤 16번 지휘_판다 복간 추천 카드 발동.
3. **2.5초 뒤** AI 헬퍼 패널에 💡 **[이지퍼비터] 인쇄 원가 최적화 제안** 카드 자동 표시.
4. 카드에서 신국판(12원/면) vs A5(8원/면) 비교표 및 총 절감액 확인.
5. **[✅ A5 판형 최적화 승인 및 시뮬레이션 가동]** 버튼 클릭.
6. 16번 지휘_판다 배너 메시지 업데이트 + 3번 에이전트 감사 로그 기록.
7. 0.5초 후 해당 도서의 시뮬레이션 모달 자동 오픈 (A5 권장 플래그 포함).
