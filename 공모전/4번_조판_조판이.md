## 1. 에이전트 개요
*   **역할**: 2번 데이터정제_다듬이가 선별한 도서 데이터를 바탕으로 원고 텍스트를 수령하여, 인간 편집디자이너의 개입 없이 단 10초 만에 인쇄소에서 즉시 출력할 수 있는 인쇄 표준 규격 PDF/X-4 파일로 조판 렌더링하는 **스펙 가변형 자동 조판(Trim & Page Adaptive Typesetting)** 에이전트.
*   **목적**: 대표님이 확립한 **'4대 표준 판형 고정 정책'**에 따라 신국판 등 4대 규격을 기반으로 도련선(Bleed 3mm), 여백(Margin), 쪽번호(Page Number), 머리말(Running Head)을 포함한 완성형 인쇄 파일을 자동 생성하고 화면 뷰어 연동 및 인쇄소 토스 파이프라인을 구축한다.
*   **핵심 철학**: "디지털 조판 비용의 제로화(Zero-labor)". 전통적인 권당 15만~30만 원 상당의 기획/조판 디자인 비용을 Gemini API와 웹 표준 CSS Paged Media 기술을 결합하여 권당 1.56달러(약 2,000원) 수준으로 낮추는 생산 비용 혁신의 중심이다.

## 3. 세부 구현 로드맵

### ✅ Step 1: 조판 정보 파라미터 수령 및 4대 표준 판형 자동 필터링
*   도서명, 저자명, 판형 규격, 종이 평량(두께), 페이지 수 수령.
*   **4대 표준 규격 강제 필터**: 아래 4대 규격 외의 예외적 판형은 접수 시 4대 규격 중 가장 가까운 판형으로 자동 변환/제한하여 조판을 진행한다.
    1.  **신국판 (152 x 225 mm)**: 학술서, 소설, 인문/사회 분야 표준 (메인 판형)
    2.  **A5 국판 (148 x 210 mm)**: 에세이, 실용서 표준
    3.  **46배판형 (188 x 257 mm)**: 시집, 에세이, 소형 소설 문고판 표준 (기존 46판에서 상향 개편)
    4.  **국배판 (210 x 297 mm)**: 대형 전공서적, 매뉴얼, 그림책 표준
*   책등 두께(Spine Width) 계산 공식 자동 적용: `(내지 평량에 따른 두께) * (총 페이지 수 / 2) + 표지 평량 두께`.

### ✅ Step 2: HTML5/CSS Paged Media 템플릿 컴파일 (4대 표준 전용)
*   도서 본문 텍스트를 문단(Paragraph) 단위로 파싱하여 4대 판형별 최적화된 HTML 조판 뼈대에 매핑.
*   W3C Paged Media `@page` CSS를 통해 여백, 도련(Bleed 3mm), 양면 인쇄용 홀짝 페이지 제침(Gutter), 머리말 및 꼬리말 쪽번호 위치 자동 계산 배치.

### ✅ Step 3: PDF/X-4 인쇄 표준 포맷 렌더링 (Serverless PDF Compiler)
*   서버리스 환경에서 HTML 레이아웃을 고해상도 PDF(300 DPI 이상, 나눔명조 한글 폰트 내장)로 변환 컴파일.

### ✅ Step 4: 클라이언트 조판 뷰어 연동 및 인쇄소 발주 대기
*   생성된 PDF를 대시보드 내 조판 확인 모달 뷰어(PDF.js 등)에 바인딩하여 대표님/고객이 승인할 수 있도록 시각적 피드백 제공.
*   최종 승인 시 11번 자동화 배포 관리자 및 인쇄소 D2P 서버로 전송 대기.
