## 2. 차주 및 차기 작업 계획 (2단계 기술 고도화 진입)

기획서 비주얼 자산이 완성됨에 따라, 발표장 시연의 신뢰도를 결정짓는 Supabase 실시간 연동 및 RAG 강화 개발 작업에 착수합니다.

1.  **대시보드 실시간 데이터 바인딩 (최우선)**
    - B2C 스토어 펀딩 클릭(결제) 발생 시 ➡️ Supabase `supporter_logs` 테이블 인서트 ➡️ control.js 실시간 Realtime 감지 ➡️ 대시보드 게이지/그래프 즉각 애니메이션 상승 연동.
2.  **16번 지휘_판다 RAG 및 JSON 파싱 버그 조치**
    - 보안 정책 및 에이전트 매뉴얼 RAG 지식 주입 보강.
    - 챗봇 프롬프트 수정 및 깨진 JSON 출력 원인 파악 후 수정.
3.  **실제 복간 타겟 도서 데이터베이스 세팅**
    - 가상 도서 ➡️ 실제 복간 도서 서지 데이터 및 커버 이미지로 스토어 DB 마이그레이션.
4.  **가변 데이터(VDP) 개인화 증서 PDF UAT**
    - 펀딩 성공 UAT 시 PDF 폰트 깨짐 방지 및 속도 점검.
