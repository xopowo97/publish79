# 출판친구 개발 및 설계 일지 (2026년 7월 초)

## 1. 듀얼 대시보드 고도화 및 실증 데이터 연동 (2026-07-01)
* **목적**: 심사위원 대상 실증력 강화를 위해 B2C 스토어와 B2B 통제실의 듀얼 대시보드에 기획서 수치와 수학적으로 연계되는 실시간 통계 보드 및 영업/마케팅 타임라인을 신설하고 데이터의 실시간 동기화를 구현한다.
* **주요 개발 내용**:
  1. **B2C 펀딩 스토어 (index.html, style.css)**:
     - 실시간 라이브 현황판 4열 그리드 이식: 구출된 종이책(누적 부수), 실시간 활성 서포터 수, 소나무 상쇄 그루, 진짜 기부 묘목 수.
     - ESG 탄소 감축 및 묘목 매칭 기부 연동 수식:
       * `소나무 상쇄 공식`: 펀딩 부수 x 0.166그루 (1부당 종이 0.5kg 절약 -> 연간 파쇄 예방량 비례 소나무 상쇄 효과 매핑)
       * `묘목 기부 공식`: 15부 결제 성공 시 강원도 삼척 피해지에 1그루 기부 조림 기금 적립 (권당 200원 적립, 묘목 1그루 3,000원 매칭)
       * `캠페인 프로그레스`: 목표 400그루 대비 달성률 동적 렌더링.
     - 펀딩 성공 UAT(마지막 50번째 클릭 결제 성공 시) 발생 시 카운트업 애니메이션(`animateValue`) 헬퍼를 통해 스토어 대시보드가 실시간 동적 갱신되도록 처리.
  2. **B2B 어드민 통제실 (control.html, control.js, control.css)**:
     - 복간 후보 TOP 3를 **TOP 5**로 확장하고 등수별 메달 이모지 및 타입 정렬 안전 가드(`Number()`, `String()` 명시적 변환) 적용.
     - 10번 B2B 영업이의 실증 출판사 영업 타임라인 피드 신설:
       * 주요 타겟 오프라인 출판사인 `상상아카데미`('마녀' 계약성공), `메디치미디어`('침묵의 봄' 협의중), `청어출판사`('국가란 무엇인가' 제안준비)의 B2B 영업 상태를 실시간 연동.
     - 11번 알리미의 AI 마케팅 배포 현황판 탑재:
       * 유튜브 Shorts, 인스타 Reels, 카카오 알림톡, B2C 포털 API 채널의 배포대기/배포완료(녹색 네온 글로우) 상태 전환. 파이프라인 가동 성공 시 자동으로 마케팅 채널들이 배포 완료 상태로 동기화 갱신되는 상호작용 탑재.
  3. **데이터 타입 가드 및 Rate Limit**:
     - Supabase 등 DB 데이터 정렬 및 매핑 시 타입 불일치로 인한 돔 크래시를 차단하기 위해 자바스크립트 레벨에서 형변환 가드를 구축.
     - 폴링 및 웹소켓 연동 시 분당 20회 API 호출 제한(429 Too Many Requests) 정책 준수 검증 완료.
